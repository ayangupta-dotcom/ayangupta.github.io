Feature Extraction, Density Estimation and Bayesian Classification

In this project used the subset of images from the MNIST dataset and this dataset contains 70,000 images of handwritten digits divided into 60,000 training images and 10,000 testing images. In performed three task first the feature extraction and normalization in this first flatten the dataset and find the mean and standard deviation of the 784 pixels as each image is stored as the 28x28 array of pixels and computed the mean and standard deviation as the two features. Then compute the mean Mj, standard deviation Sj where j=1,2 for the first and second feature of the 2d vector and computed the normalization Yi . Then the second task performed was the desity estimation for this used the MLE method to estimate the parameters for the 2-d normal distribution for each class/digit, using the respective training data for that class/digit. Then the third task the Bayesian Decision Theory for optimal classification in this did the estimated distribution for finding the minimum error-rate classification.
